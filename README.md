# ChemBot Research Lab

## 项目简介

本项目是《仿真至现实闭环：复杂时空场景下具身智能关键问题研究》的项目主页，致力于研发一套前沿的"具身智能机器人化学家"系统，旨在突破当前化学实验智能化方面的核心瓶颈。

我们旨在创造一位能够与科研人员无缝协作的"智能研究伙伴"，它将深度融合化学、人工智能、机器人与物联网技术。该系统具备从理解到执行再到学习的完整闭环能力。我们的最终愿景是打造"无人实验室"的核心雏形，将化学家从繁重重复的体力劳动中彻底解放出来，让他们能真正回归科学发现与知识创造的本源。

## 核心特性

- **项目展示**：展示研究组在具身智能、机器人操作、强化学习等领域的前沿研究
- **团队介绍**：展示研究组成员及其研究方向
- **研究成果**：发布最新的学术论文和研究成果
- **AI 助手**：集成基于 Google Gemini 的智能对话助手 ChemBot，可回答关于研究组、项目和成员的相关问题
- **新闻动态**：实时更新研究组的最新动态、获奖信息和学术活动
